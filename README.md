# shopHouse

Online shopping web app where members can purchase furniture and also sign up to be sellers to sell their furniture.

> Keywords: REST API, Spring Boot, React, JWT authentication

## Contribution

- Personal project

## Tech stack

### Frontend

- JavaScript ES6
- React v18.2
- Vite v4.0
- Redux-Toolkit v1.9.1

### Backend

- Spring Boot v2.7.7
- Java 11
- Spring Data JPA
- H2 Database v2.1
- Gradle
