# shopHouse

> Keywords: REST API, Spring Boot, React, JWT authentication

- Vite v4.0
- Redux-Toolkit v1.9.1

### Backend

- Spring Boot v2.7.7
- Java 11
- Spring Data JPA
- H2 Database v2.1
- Gradle
